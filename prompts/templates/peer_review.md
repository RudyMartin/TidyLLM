# Peer Review Workflow Template
*Professional peer review and expert validation workflow*

## Peer Review Document Assessment
```
Peer Review Initial Assessment:

Conduct initial peer review assessment to establish review parameters:

Document for Peer Review: {document_content}

Initial Assessment:
1. **Document Scope and Purpose**:
   - Document type and subject matter
   - Intended audience and stakeholders
   - Business or academic context
   - Review objectives and criteria

2. **Content Complexity Analysis**:
   - Technical complexity level
   - Subject matter expertise required
   - Interdisciplinary considerations
   - Industry-specific knowledge needs

3. **Review Framework Setup**:
   - Review criteria and standards
   - Evaluation methodology
   - Peer reviewer qualifications needed
   - Timeline and milestone requirements

4. **Quality Baseline Establishment**:
   - Current quality indicators
   - Benchmark standards applicable
   - Previous version comparisons
   - Industry best practice alignment

Output peer review framework with assessment criteria and methodology.
```

## Subject Matter Expert Review
```
Subject Matter Expert Analysis and Validation:

Detailed expert review focusing on content accuracy and professional standards:

Content for Expert Review: {document_content}

Expert Analysis:
1. **Technical Accuracy**:
   - Factual correctness verification
   - Methodology validation
   - Data and calculation accuracy
   - Industry standard compliance

2. **Professional Standards**:
   - Best practice adherence
   - Professional ethics compliance
   - Industry regulation alignment
   - Certification/accreditation requirements

3. **Content Depth and Coverage**:
   - Comprehensive topic coverage
   - Appropriate level of detail
   - Missing critical elements
   - Scope and boundary appropriateness

4. **Innovation and Value Assessment**:
   - Novel approaches or insights
   - Value proposition clarity
   - Competitive advantage identification
   - Implementation feasibility

Provide expert validation with specific technical recommendations and corrections.
```

## Comparative Analysis and Benchmarking
```
Peer Comparative Analysis:

Compare document against peer documents, industry standards, and best practices:

Document for Comparison: {document_content}

Comparative Analysis:
1. **Peer Document Comparison**:
   - Similar document analysis
   - Competitive document benchmarking
   - Industry leader comparison
   - Academic or research standard alignment

2. **Performance Benchmarking**:
   - Key performance indicator comparison
   - Success metric benchmarking
   - Efficiency and effectiveness measures
   - Return on investment analysis

3. **Best Practice Identification**:
   - Industry best practice alignment
   - Innovation and differentiation factors
   - Improvement opportunity identification
   - Competitive gap analysis

4. **Recommendation Development**:
   - Performance improvement recommendations
   - Best practice adoption suggestions
   - Competitive positioning strategies
   - Innovation and differentiation opportunities

Format as comprehensive benchmarking report with peer comparison matrix.
```

## Peer Review Quality Assurance
```
Peer Review Quality Validation and Approval:

Final quality assurance review with peer validation and approval recommendations:

Complete Review Package: {document_content}

Quality Validation:
1. **Review Completeness**:
   - All review criteria addressed
   - Expert feedback incorporation
   - Comparative analysis completion
   - Quality standard achievement

2. **Peer Consensus Assessment**:
   - Multiple reviewer agreement
   - Conflicting opinion resolution
   - Consensus building outcomes
   - Final recommendation alignment

3. **Professional Validation**:
   - Professional standard certification
   - Industry expert endorsement
   - Regulatory compliance confirmation
   - Stakeholder acceptance validation

4. **Final Recommendations**:
   - Approve for publication/implementation
   - Approve with minor revisions
   - Major revision requirements
   - Reject with specific feedback

Present as peer review certification with clear approval status and professional validation.
```
