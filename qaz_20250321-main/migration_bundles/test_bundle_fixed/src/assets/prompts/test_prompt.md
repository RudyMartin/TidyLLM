# QA Review Prompt

This is a test prompt file for QA document processing.

## Instructions

Please analyze the uploaded documents and provide a comprehensive QA review focusing on:

1. **Document Completeness**: Ensure all required sections are present
2. **Data Quality**: Validate data accuracy and consistency
3. **Compliance**: Check regulatory compliance requirements
4. **Risk Assessment**: Evaluate potential risks and mitigation strategies

## Specific Requirements

- Use the regular QA orchestrator for processing
- Generate detailed findings and recommendations
- Provide risk scores and compliance assessments
- Include actionable next steps

## Expected Output

- QA assessment report
- Risk evaluation matrix
- Compliance checklist results
- Recommendations for improvement
