### 🔍 **AUTOMATIC SABOTAGE DETECTION**

1. **File Count Limits**: Max 10 files (auto-truncates)
2. **Size Limits**: 50MB per file, 200MB total (auto-filters) 
3. **Suspicious Filename Detection**: 
   - `.exe`, `.zip`, `.tar`, `.gz`

4. **Memory Protection**: MemoryError handling with graceful fallback

### 🚨 **EMERGENCY CONTROLS**

#### Emergency Demo Mode (Sidebar)
- **Button**: "🚨 Emergency Demo Mode" 
- **Function**: Bypasses all file processing with mock data
- **Use**: When teammates completely break file uploads

#### Demo Reset (Sidebar) 
- **Button**: "🔄 Reset Demo"
- **Function**: Clears all session state, returns to Step 1
- **Use**: When demo gets stuck or corrupted
