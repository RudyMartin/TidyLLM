#### **Financial Risk Management Tabs – Summary for Demo**
Added new tabs to support model governance and documentation review:

- **Plan Audit**  
  Uses DSPy to compare Model Development Plans against structured ground truth. Flags gaps and shows side-by-side comparisons.

- **Truth Synthesizer**  
  Helps generate or refine audit-ready ground truth records from uploaded PDFs. These are stored and embedded for reuse.

- **Audit Log Viewer**  
  Semantic search over prior audit logs stored in FAISS, with timestamps and key metadata.

- **Vector Diagnostics**  
  Backend tool to check FAISS and PgVector health, embedding versions, and indexing status.

These tools improve traceability, compliance, and review workflows—especially useful for regulated environments.
