 Standard Benchmark Datasets for Robot Manipulation

  Yes, there IS a "Kaggle" equivalent! Teams without robots use these:

  | Dataset                                                      | Source      | Images      | Notes                        |
  |--------------------------------------------------------------|-------------|-------------|------------------------------|
  | https://huggingface.co/datasets/HuggingFaceVLA/libero        | HuggingFace | 273K frames | Standard benchmark, 40 tasks |
  | https://huggingface.co/datasets/openvla/modified_libero_rlds | HuggingFace | RLDS format | Used for OpenVLA fine-tuning |
  | https://github.com/stepjam/RLBench                           | Simulation  | 100 tasks   | Infinite demos via sim       |
  | Bridge V2                                                    | Berkeley    | 124GB       | Real WidowX robot data       |
