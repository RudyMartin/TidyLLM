# Demo Scripts

Demonstration and showcase applications for TidyLLM platform capabilities.

## System Demonstrations

### **heiros_demo.py**
Complete Heiros system demonstration:
- Shows end-to-end audit workflow processing
- Demonstrates FLOW agreement execution
- Displays corporate compliance reporting
- Interactive demo with real data examples

### **demo_file_upload_app.py**
File upload and processing demonstration:
- Interactive file upload interface
- Real-time document processing visualization
- Embedding creation and storage demo
- S3 integration showcase

### **demo_status_check.py**
System status monitoring demonstration:
- Live system health dashboard
- Database connectivity demonstrations
- AWS service status visualization
- Real-time metric monitoring

### **final_demo_status_report.py**
Comprehensive demo status reporting:
- Generates complete system capability report
- Shows all integrated components working
- Produces presentation-ready status summary
- Validates demo environment readiness

## Usage

### Run Interactive Heiros Demo
```bash
python scripts/demos/heiros_demo.py
```

### Launch File Upload Demo
```bash
python scripts/demos/demo_file_upload_app.py
```

### System Status Demo
```bash
python scripts/demos/demo_status_check.py
```

### Generate Demo Report
```bash
python scripts/demos/final_demo_status_report.py
```

These demonstrations showcase the full capabilities of the TidyLLM platform in interactive, presentation-ready formats suitable for stakeholder reviews and system validation.
